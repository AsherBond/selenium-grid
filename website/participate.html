Hack Some Code

We welcome new contributors. The project is still at an very early stage and you can have significant impact. Here is the best way to start contributing:

  1. Install Java 5+ and Ant 1.7.x
  2. Make sure that they are in your PATH : java -version , ant -version
  3. svn co https://svn.openqa.org/svn/selenium-grid/trunk <project root>
  4. cd <project root>
  5. Run the build, to make sure that it passes on your box: ant dist
  6. Hack some cool features and do not forget to write some tests.
  7. Run the build again, make sure that it still passes.
  8. Create a patch: svn diff > patch_to_fix_this_and_that.diff
  9. Send us the patch by email
