Contributing to Selenium Grid

Requirements

To run Selenium Grid you will need

  • Java 5+ JDK
  • Ant 1.7.x

Install these in your machine if you do not have them already. Do not forget to add them to you PATH.

Getting the code

svn co https://svn.openqa.org/svn/selenium-grid/trunk

Building all artifacts

  1. cd <project root>
  2. ant dist

Building a sub-project

  1. cd <project root>/hub
  2. ant package-standalone
