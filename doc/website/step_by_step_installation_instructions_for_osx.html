Step by Step Installation Instructions for Mac OS X

Here are detailed, step by step instructions on how to install Selenium Grid, on an OS X machine from Apple.

Step 1: Verify That JDK 1.5 is Installed

The appropriate JDK is shipped by default on any Mac 10.X installation, and should be available out-of-the-box. You can double-check this by openning a terminal window and running java -version

The command output should indicate that a 1.5 version of the JDK is available. If not, insert Apple OS X install CD in your machine and re-install the developer tools.

Step 2: Install Ant 1.7

Check that you have the right Ant version (1.7) by running in a terminal: ant -version

Most likely you are all set if you are running OS X Leopard: Ant 1.7 is provided as part of the default install.

If you are running an earlier version of Mac OS X, and have a 1.6 Ant version (or earlier), install Ant 1.7 as explained in step 2a.

Step 2a: Installing Ant 1.7

  1. Download Ant version 1.7 from http://apache.mirrors.hoobly.com/ant/binaries/apache-ant-1.7.0-bin.zip
  2. Once it has been downloaded, unzip the file in a directory on your local system. For instance under /Applications/Java/apache-ant-1.7.0:

  3. You then need to modify your PATH so that your system finds first the freshly installed ant launcher.

    You can do this by going to a Terminal window and:

    1. Navigate to your home directory (just type cd and then the ENTER key), and open the .profile file.
    2. Edit the .profile file so that the freshly installed Ant version 1.7 is first in your path:
    3. Save the .profile file (Menu "File"->"Save")
    4. Launch a new terminal
    5. Verify that your PATH is set accordingly and that you pick up the right Ant version:
    6. From now on, execute all commands from this fresh terminal window
